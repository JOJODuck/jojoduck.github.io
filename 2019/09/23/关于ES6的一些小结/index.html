解构赋值

let a =1; let b =2; let c =3;可以用es6写法:let [a,b,c] =[1,2,3];
括号左右两边的结构必须一样。
可以在左边设置默认值。 let [fpp=true]=[];
也可以用对象进行解构赋值： let {foo,bar}={foo:'szb',bar:'bzs'};
如果先定义了一个变量但是没有赋值，后面才进行赋值操作。需要加上圆括号：
let foo; ({foo}={foo:'szb'});
